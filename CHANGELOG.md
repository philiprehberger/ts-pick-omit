# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-03-21

### Added

- `pick()` for type-safe shallow key selection
- `omit()` for type-safe shallow key exclusion
- `deepPick()` for nested dot-notation path selection
- `deepOmit()` for nested dot-notation path exclusion
- `pickBy()` for predicate-based key selection
- `omitBy()` for predicate-based key exclusion
